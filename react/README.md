# 🏏 IPL Auction Intelligence: State-of-the-Art Cricket Analytics

A professional-grade, data-driven intelligence console designed to decode the intersection of **Auction Market Value** and **On-Field Performance**. This platform transforms over 18 years of IPL statistics (2008–2026) into actionable insights using advanced visualization and high-fidelity archival data.

## 🚀 Key Features

### 1. 🏛️ Complete Franchise Archives (15 Teams)
The platform now features a comprehensive historical record for all 15 franchises, including defunct teams like **Deccan Chargers**, **Kochi Tuskers Kerala**, **Rising Pune Supergiant**, and **Gujarat Lions**.
*   **Management Audit**: Documented historical coaching staff, mentors, and directors for every team.
*   **Squad Intelligence**: Precise player roles, styles, and nationalities across all active and inactive seasons.

### 2. 🌌 Market Galaxy (Auction ROI Maps)
A high-performance scatter plot mapping every player's **Auction Price** against their **Value Score**.
*   **ROI Benchmarking**: Real-time calculation of performance-per-crore spent.
*   **Star Highlighting**: Record-breaking signings (e.g., Auction 2025) are highlighted to visualize their impact on team budgets.

### 3. 🧠 Strategy IQ (Efficiency Index)
A team-level analytics engine that ranks franchises by their **Spending Efficiency**. 
*   **Bargain Detection**: Identifies teams that consistently secure high-performance assets at low prices.
*   **Budget Optimization**: Visualizes how much value a team derives from every Rupee spent in the auction.

### 4. 📈 Performance Pulse (Trend Mapping)
Tracks player trajectories across multiple seasons, allowing for "slump vs. surge" detection and career-peak identification.

### 5. ⚡ JSON-Driven Static Architecture
Optimized for high-speed delivery on **Vercel**. The analytical kernel has been migrated from a Python runtime to a pre-processed JSON-bundle architecture, ensuring sub-second load times for complex visualizations.

## 🛠️ Technology Stack
*   **Core Logic**: Javascript / Vanilla CSS (High-contrast UI)
*   **Visuals**: Chart.js 4.4.2 (Interactive ROI & Efficiency charts)
*   **Data Pipeline**: Unified JSON-Bundle System (cross-module compatibility)
*   **Infrastructure**: Vite + Vercel (Clean-URL routing)
